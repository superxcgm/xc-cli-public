# XC CLI Public

## Install

### Requirement
* JRE(Java Runtime Environment)

### Pre Build
Download the latest release, unzip, change directory, and run `make install`.
Then add `export PATH="$HOME/local/bin:$PATH"` to your `.zshrc` or `.bashrc`.

### From Source Code
Not opensource now.

## Usage
Run `f` to get help.

## Upgrade
Run `f upgrade`
